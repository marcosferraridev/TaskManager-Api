\# Task Manager API



REST API developed with ASP.NET Core for task management.



This project was created for academic and learning purposes, focusing on the practical study of backend development concepts using C# and ASP.NET Core Web API.



The main goal of this project is to progressively evolve the application while learning new technologies and backend development practices. Future improvements may include database integration, authentication with JWT, Entity Framework, Docker, Clean Architecture, automated tests, and deployment to cloud platforms.



\---



\# Technologies Used



\* C#

\* ASP.NET Core Web API

\* Swagger / OpenAPI

\* Git \& GitHub



\---



\# Running the Project



\## Clone the repository



```bash

git clone https://github.com/marcosferraridev/TaskManager-Api.git

```



\---



\## Navigate to the project folder



```bash

cd TaskManager-API

```



\---



\## Run the application



```bash

dotnet run

```



\---

\## Swagger



After running the project, Swagger will be available at:



https://localhost:{port}/swagger
